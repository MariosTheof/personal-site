A full-stack web application that tracks PC component prices across multiple eBay marketplaces, helping users find the best deals on computer hardware.

## Key Features

- **Multi-marketplace tracking** - Monitors prices across different eBay regions
- **Intelligent filtering** - 96% accuracy classifier for filtering irrelevant listings
- **Real-time data pipeline** - Processes 10,000+ daily listings automatically
- **Advanced caching** - Multi-level caching strategy reducing API costs by 70%
- **Responsive design** - Built with Vue.js and Tailwind CSS

## Technical Implementation

- **Frontend**: Vue.js with Tailwind CSS for a modern, responsive interface
- **Backend**: Python-based data pipeline with automated scraping and processing
- **Infrastructure**: Initially self-hosted, migrated to Oracle Cloud for scalability
- **Data Processing**: Custom algorithms for price analysis and trend detection
- **Caching**: Strategic caching at API, exchange rates, and shipping data levels

This project demonstrates expertise in full-stack development, data engineering, and cloud infrastructure management.
