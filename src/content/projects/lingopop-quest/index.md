A innovative browser extension that transforms casual web browsing into an opportunity for continuous language learning through strategic vocabulary popups and spaced repetition.

## Core Functionality

- **Seamless Integration** - Non-intrusive language learning during normal browsing
- **Multiple Languages** - Support for Spanish, French, Polish, Russian, Italian, German
- **Smart Timing** - Vocabulary prompts every 20 minutes for optimal retention
- **Progress Tracking** - Comprehensive analytics on learning progress and frequency
- **Spaced Repetition** - Algorithm-based vocabulary reinforcement for long-term retention

## Technical Architecture

- **Frontend**: HTML, CSS, JavaScript for browser extension interface
- **Backend**: Laravel framework handling user data and vocabulary management
- **Database**: CockroachDB for distributed, scalable data storage
- **Infrastructure**: Google Cloud Platform with auto-scaling configuration
- **Deployment**: Terraform for infrastructure as code
- **Performance**: Supports 2M+ queries per month across 100+ instances

## Learning Features

- **Bite-sized Quizzes** - Interactive vocabulary exercises
- **Cultural Context** - Contextual learning with real-world examples
- **Personalized Paths** - Adaptive learning based on individual progress
- **Premium Tier** - Ad-free experience with advanced features

This project showcases skills in browser extension development, distributed systems, and scalable cloud architecture.
